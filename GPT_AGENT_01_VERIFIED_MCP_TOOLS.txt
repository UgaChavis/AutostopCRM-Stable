Key usage notes
- For broad operational context, call review_board first.
- For full hidden board context and recent history, call get_gpt_wall.
- For mass movement between columns, use bulk_move_cards, not many move_card calls.
- rename_column changes only the label and keeps the same column id.
- delete_column works only for empty columns.
- create_cash_transaction is the main write operation for cashboxes.

Critical working rule
- This toolset is for the current AutoStop CRM board only.
