AUTOSTOP CRM / GPT AGENT OPERATING PLAYBOOK

Verified on: 2026-04-07

START OF EVERY NEW CHAT
1. Call ping_connector
2. Call bootstrap_context
3. If anything looks wrong, call get_runtime_status
4. Confirm:
- connector_name
- resource_url
- board_name
- scope_rule

BOARD REVIEW FLOW
1. review_board for manager summary
2. get_gpt_wall if full context or recent history is required
3. get_card or get_card_log for exact card details

CARD CREATION FLOW
1. If the user gives free text, normalize the short essence into title
2. Keep vehicle limited to make/model
3. Put the detailed issue into description
4. If needed, call autofill_vehicle_data before create_card

CARD UPDATE FLOW
1. Use get_card first if there is any ambiguity
2. Use update_card for content changes
3. Use move_card for one card
4. Use bulk_move_cards for many cards
5. Use archive_card and restore_card for lifecycle changes

STICKY FLOW
1. create_sticky for a new note
2. update_sticky to edit text or deadline
3. move_sticky to reposition
4. delete_sticky to remove

CASHBOX FLOW
1. list_cashboxes
2. get_cashbox for current totals and history
3. create_cash_transaction for income or expense
4. create_cashbox or delete_cashbox only when explicitly asked

REPAIR ORDER FLOW
1. Read the card first with get_card
2. If broad board context matters, read get_gpt_wall first
3. Use card updates carefully when changing repair-order-related content
4. Payment logic in the UI is tied to cashbox choice:
- cashbox "Безналичный" adds +15 percent taxes and fees for that cashless payment amount
- cashbox "Наличный" does not add taxes
- cashbox "Карта Мария" does not add taxes

SAFETY RULES
- Do not operate on any other board or connector
- Do not guess identifiers if they can be read first
- For broad edits, read context first and then write
- After major write operations, read back the card or board summary to verify the result

SERVER WORKFLOW FOR HUMAN OPERATOR
- Local code change
- git commit
- git push origin autostopCRM
- On server:
  cd /opt/autostopcrm
  ./deploy.sh
